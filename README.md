# annotated-etext-react

export code with `babel` before pushing to repo:

```yarn run babel src/ -d lib```

import into `react` as usual:

```import AnnotatedEtextContainer from 'annotated-etext-react';```

To run the demo:

```
cd demo
yarn install
yarn start
```
